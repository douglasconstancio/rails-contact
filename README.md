## ⬇️ Como baixar

```bash
  // Clonar repositório
  $ git clone https://github.com/douglasconstancio/rails-contact.git

  // Inicie o banco de dados
  $ rails db:migrate

  // Caso queira ver as rotas
  $ rails routes

  // Rodar o projeto
  $ rails s
```
